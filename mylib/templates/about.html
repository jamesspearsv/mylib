About MyLib

Lib was created as a final project idea for Harvard's CS50x 2021. The project utilizes HTML, CSS, CSS Bootstrap, Javascript and Python to create a personal cataloging web app for your home library collection.

The project is inspired by and draws heavily on various book releted web resources including LibraryThing, Goodreads, and Google Books. I'd highly enourage you to check those out services if you are interested.

Data for this web app is drawn from the Google Books API. You can check out its documentation here. Other APIs that were helpful during the development process included the Open Library API.

Feel free to check out the GitHub repo for this project.

Sign up for an account or login to try it out.
